Web crawlers

Web crawlers (also known as spiders, bots and automatic indexers) are computer programs that scan the Web, ‘reading’ everything they find. These crawlers scan web pages to see what words they contain, and where those words are used. The crawler turns its findings into a giant index. The index is basically a big list of words and the web pages that feature them. So when you ask a search engine for pages about IB Computer Science, the search engine checks its index and gives you a list of pages that mention IB Computer Science. Web crawlers scan the web regularly so they always have an up-to-date index of the web.

Parallel web crawlers

A parallel crawler is a crawler that runs multiple processes in parallel. The goal is to maximize the download rate to avoid repeated downloads of the same page. To avoid downloading the same page more than once, the crawling system requires a policy for assigning the new URLs discovered during the crawling process, as the same URL can be found by two different crawling processes. Parallel web crawlers might index the same page multiple times.

Meta tags

Meta tags are the words that are hidden in your code. People browsing your site will just not be able to see them. It is a way of hiding words in your webpages for search engines. The search engines still read them for it to have a simple, summarized idea of what your site is about and what exactly your keywords are. The importance of meta tags is that the search engines read them in order to compare if these keywords and the description are related with the visible content.

robots.txt

A robots.txt file tells search engine crawlers which pages or files the crawler can or can't request from your site. This is used mainly to avoid overloading your site with requests. After arriving at a website but before crawling it, the search crawler will look for a robots.txt file. If it finds one, the crawler will read that file first before continuing through the page. Because the robots.txt file contains information about how the search engine should crawl, the information found there will instruct further crawler action on this particular site. If the robots.txt file does not contain any directives that disallow a user-agent’s activity (or if the site doesn’t have a robots.txt file), it will proceed to crawl other information on the site.

Web indexing

Web indexing is the process of adding webpages into Google search and the processing of the information gathered by the bots from thier crawling activities. Depending on which meta tags used, Google will crawl and index your pages. A no-index tag means that that page will not be added into the web search's index.
